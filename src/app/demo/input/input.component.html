文本输入框

文本输入框，用于日常表单的数据输入

姓名 别名(可选) 密码 数目 禁用 只读 单位组合 网址 搜索框 搜索框

使用方式:

参数

参数 说明 默认值
type 输入类型 text
placeHolder 输入提示 请输入
modelValue 双向绑定值 ''
required 是否必选,true/false true
disabled 是否禁用,true/false false
readon 是否只读,true/false false
isSearch 是否搜索,true/false false
prefixList 前缀单位列表 []
suffixList 后缀单位列表 []
min 最小值 0
max 最大值 0
width 自定义宽度，px或百分比，比如：50%，400px 470px
