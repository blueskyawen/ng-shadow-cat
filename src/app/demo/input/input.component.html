参数

参数 说明 默认值
type 输入类型 text
placeHolder 输入提示 请输入
modelValue 双向绑定值 ''
required 是否必选,true/false true
disabled 是否禁用,true/false false
autofocus 是否自动聚焦,true/false false
readon 是否只读,true/false false
isSearch 是否搜索,true/false false
prefixList 前缀单位列表 []
suffixList 后缀单位列表 []
min 最小值 0
max 最大值 0
step 步长 1
width 自定义宽度，px或百分比，比如：50%，400px 100%
height 自定义高度，px或百分比，比如：50%，400px 36px
iconffix 是否带前后缀图标，pre/sub
showHint 是否聚焦后显示提示 true
noBorder 是否不带边框，false/true false
inputLintText 提示内容 请输入中文或者英文
inputErrorText 错误提示内容
showError 是否提示错误，false/true false
pattern 检查正则对象
