正常选择
禁止选择

使用方式:

参数

参数 说明 默认值
options select项目列表 []
selections 默认选择项列表 []
isDisable 是否禁用,true/false false
hasheckbox 是否带复选框,true/false false
allcheck 是否带全选框,true/false false
showCheckNum 是否显示选择数目,true/false false
