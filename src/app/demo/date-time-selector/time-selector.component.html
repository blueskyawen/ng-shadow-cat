参数 说明 默认值
type 类型,single/input single
date 日期对象 object()
disableHours 禁止选取的Hour列表 []
disableMinutes 禁止选取的Minute列表 []
disableSeconds 禁止选取的Second列表 []
width 自定义宽度，px或百分比，比如：50%，400px 100%
ncFormat 自定义显示格式 hh:mm:ss
