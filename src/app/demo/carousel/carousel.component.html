轮播

轮播，一种灵活的响应式的向站点添加滑块的方式

1. 左右滑动

2. 渐显

3. 自定义尺寸

4. 启动/停止自动播放

自动播放

鼠标放置停止播放

5. 轮播切换事件(输出当前显示图的索引)

使用方式:

参数

参数 说明 默认值
captions 轮播项目数组 []
timerlen 自动播放时长,默认不设为undefined,不自动播放 undefined
effect 轮播类型，scroll/fade scroll
width 自定义宽度，string 100%
height 自定义高度，string 650px
isPause 鼠标浮动其上是否停止自动播放，离开恢复播放，Boolean false
